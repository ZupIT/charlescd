# CharlesCD's UI components documentation
<br/>
<br/>

## Overview

This documentation shows all CharlesCD's existing UI components that works as building blocks to compose our user's interfaces. You will find here all information related to the code, description and usage of each component. 

## How to use

You can browse through the components in the sidebar, and click on `show code` to see its usage. You can also see the specification of each component.

## Contribute

You are able to create a new component or extend existing ones. Check out our [**contribution guide**](https://github.com/ZupIT/charlescd/blob/main/CONTRIBUTING.md) to see how you can be part of CharlesCD's library.<br/>
Note: Before creating a component, please verify if there is one that already works for what you need.
